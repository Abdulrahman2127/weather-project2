
# 🌤️ Saudi Weather Hub

A simple weather application built with React that displays live weather information for major cities in Saudi Arabia using the OpenWeather API.

## ✨ Features

- 🌍 Live weather data
- 🌡️ Current temperature
- 📉 Minimum & Maximum temperature
- ☁️ Weather description with icons
- 🌐 Arabic & English language support
- 📱 Responsive design
- ⚛️ Built with React and Material UI

## 🛠️ Technologies

- React
- Vite
- Axios
- Material UI
- Moment.js
- React i18next
- OpenWeather API

## 🚀 Installation

```bash
git clone https://github.com/YourUsername/weather-project.git

cd weather-project

npm install

npm run dev
```
